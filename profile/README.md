# AirShelf

**Semantic commerce infrastructure for the AI era.**

We help e-commerce brands connect with customers through AI-powered shopping assistants. Our platform translates natural language shopping intents into product recommendations using semantic search and intelligent matching.

## What We Build

- **Semantic Search API** — Intent-based product discovery for AI agents
- **BIME** — Buy Intent Matching Engine for Telegram and other platforms
- **Merchant Tools** — Analytics, benchmarking, and optimization dashboards
